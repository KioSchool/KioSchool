# Claude Code Instructions

공통 컨벤션은 AGENTS.md를 참조한다. 이 파일에는 Claude Code 전용 설정만 포함한다.

Read AGENTS.md for project conventions, tech stack, naming rules, component patterns, and architecture.

## Language

사용자와의 대화는 한국어로 한다. 코드, 변수명, 기술 용어는 원문 그대로 유지한다.

## Commit Convention

- 커밋 메시지는 한국어로 작성
- 사용자가 명시적으로 요청하기 전까지 커밋/푸시하지 않는다
